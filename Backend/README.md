# Mon vieux Grimoire - Backend

Ce dépôt contient le backend de l'application Mon Vieux Grimoire : un site web de référencement et de notation de livres.

## Comment lancer le projet ? 

### Avec npm

Faites la commande `npm install` pour installer les dépendances puis `npm start` pour lancer le projet. 
Ou si vous preferez, vous pouvez lancer le projet avec `nodemon` qui applique les changements que vous pouvez faire et relance directement le serveur.

Le projet a été testé sur node v21.1.0.
